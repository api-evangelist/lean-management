# Lean Management (lean-management)
Lean Management is the discipline of planning, organizing, and overseeing lean-related activities to achieve optimal outcomes. Drawing on the Toyota Production System and the broader lean manufacturing tradition, lean management emphasizes continuous improvement (kaizen), respect for people, elimination of waste (muda), value-stream thinking, and pull-based flow. In a software and API context, lean management informs how product, platform, and operations teams plan, ship, and improve digital capabilities, and is a foundational input for adjacent disciplines such as DevOps, agile delivery, and platform engineering.
